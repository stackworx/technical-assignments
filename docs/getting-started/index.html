Getting Started

Resources#

  • Source Tree Tutorial
  • Git Introduction if you prefer to use the CMD instead of source tree

Notes: Git is not a requirement for completing the assignment but it is recommended. When reviewing the assignment we would like to go through the commits to understand the thinking. So if you do use git please commit often.

Please create an account on either Gitlab, GitHub or Bitbucket and place your repository there.

You should have been asked to complete one of the following technical assignments#

  • ReactJs
  • NodeJs
  • .Net Core
  • Python
  • SQL

IDE#

We recommend using VSCode but you are free to use the IDE of your choice.

Questions#

If you have any questions please email assignment@stackworx.io
